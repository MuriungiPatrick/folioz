Quick Start Guide

Getting Started in 5 Minutes

1

Download and Extract

After purchasing, download the template files and extract the ZIP archive to your desired location.

2

Open the Main File

Locate the index.html file in the main folder and open it in a code editor like VS Code, Sublime Text, or Atom.

3

Customize Content

Update the text, images, and links in the HTML file to match your brand and content needs.

4

Test Locally

Open the index.html file in your web browser to preview your changes.

5

Upload to Server

Once satisfied, upload all files to your web hosting server via FTP or your hosting provider's file manager.

Color Scheme

Customizing Colors

Folioz uses CSS variables for easy color customization. The main color scheme can be modified by updating these variables in the :root selector at the top of the CSS file.

Primary #7D56FD
secondary #081828
Text #4c4c4c
Body bg #fafafa

To change the color scheme, locate this code in the CSS file:

:root {
                --primary-color: #7D56FD;
                --secondary-color: #081828;
                --text: #4c4c4c;
                --body-bg: #fafafa;
                --white: #ffffff;
            }

Simply replace the hex codes with your preferred colors, and the changes will apply throughout the entire website.
